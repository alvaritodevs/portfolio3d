# 3D Developer Portfolio (Static)

This is a lightweight, static **3D developer portfolio** built with:

- **HTML/CSS** for the layout and UI
- **Three.js (CDN via import maps)** for the interactive 3D background (`main.js`)

## Run locally

Any static server works. For example:

```bash
python3 -m http.server 5173
```

Then open `http://localhost:5173`.

## Customize

- **Name / links / email**: edit `index.html`
- **Projects**: edit the “Projects” cards in `index.html`
- **3D nodes (skills)**: edit `skillDefs` in `main.js`
